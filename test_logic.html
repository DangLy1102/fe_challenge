📝Problem Requirements

Create a function processWithDelay(numbers) that demonstrates understanding of modern async patterns.

  • The function should process each number in the array with a 1-second delay between each number. The function should print each number to the console when it is processed.
  • The function should return a Promise that resolves after all numbers are processed.
  • If the array is empty, resolve immediately.
  • If the input is not a valid array of numbers, reject with an error.

💡Code Explanation

The implementation involves the following key concepts:

  • Promise: Ensures asynchronous processing and allows success or error handling.
  • setTimeout(): Creates the 1-second delay between each number output.
  • async/await: Used for waiting between outputs in a clean, readable way.
  • reject(): Used to throw an error if the input is invalid.

📌Example Input & Expected Output

📥 Input: processWithDelay([1, 2, 3, 4, 5])
📤 Output:
(After 1 second) 1
(After 2 seconds) 2
(After 3 seconds) 3
(After 4 seconds) 4
(After 5 seconds) 5
✅ "All numbers processed"
